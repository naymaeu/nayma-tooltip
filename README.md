# Usage

Examples od use:

1. Default with tooltip on top:
```
<span ntt="Tooltip top">Top</span>
```

2. Left side:
```
<span ntt="Tooltip left" flow="left">Left</span>
```

3. Right side:
```
<span ntt="Tooltip right" flow="right">Right</span>
```

4. Down side:
```
<span ntt="Tooltip down" flow="down">Down</span>
```
